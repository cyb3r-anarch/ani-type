## Usage

### Install
`git clone <repo-url>`
`cd <repo-name>`

### to run:

`make`
`./pptux`

### Customization

You can customize the wordlist used in the typing test by navigating to `resources/wordlists/english` and modifying the words in the file. **Each word should be on a separate line with a space at the end**.
